# 💼 Portfólio Pessoal - Thiago Felizardo

Este é o meu portfólio pessoal desenvolvido com React e Vite. Aqui você encontrará minhas principais habilidades, projetos desenvolvidos, informações sobre minha trajetória profissional e links para contato.

---

## 🚀 Preview

🔗 **Acesse a versão online aqui:**  
👉 https://portifoliothiago-dev.vercel.app/

---

## ✨ Funcionalidades

- 👤 Seção de apresentação pessoal com descrição profissional.  
- 🛠️ Habilidades técnicas organizadas por tecnologias.  
- 📂 Lista de projetos com descrição e links para visualização.  
- 📄 Link para download do currículo.  
- 📬 Seção de contato com links diretos para redes profissionais.  
- 🔗 Integração com GitHub e LinkedIn.  
- 📱 Layout responsivo para dispositivos móveis e desktops.

## 🛠️ Instalação e uso local

# 1. Clone o repositório
git clone https://github.com/tfelizardo/portifolio-thiago

# 2. Acesse a pasta do projeto
cd portifolio-thiago

# 3. Instale as dependências
npm install

# 4. Inicie o servidor de desenvolvimento
npm run dev
---

## 🌎 Deploy

O projeto está hospedado na Vercel, com deploy automático integrado ao GitHub.
